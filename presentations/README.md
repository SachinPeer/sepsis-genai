# Presentations

All PowerPoint decks for the Sepsis GenAI project.

## `current/` — live, shareable decks

| File | Purpose |
|---|---|
| `Sepsis_GenAI_DeepDivev6.pptx` | Main technical deep-dive deck (latest, v6). Shared with internal / clinical audiences. |

| `Sepsis_GenAI_HospitalConfig.pptx` | Hospital-specific guardrail configuration story. |
| `Medbeacon_Executive_Overview.pptx` | Generated by `docs/generate_pptx.py` — executive / leadership overview. |

## How decks are generated

Two of the Medbeacon decks are produced by code (regenerable):

```bash
python docs/generate_pptx.py            # → presentations/current/Medbeacon_Executive_Overview.pptx
python docs/generate_storyboard.py      # → presentations/current/Medbeacon_Clinical_Storyboard.pptx
```

The other decks are hand-authored in PowerPoint.
